# 🛠️ ShopZoon - Full-Stack E-commerce Server

This is the robust backend API for ShopZoon, built with **Node.js, Express, and MongoDB**. It handles secure authentication, payment processing, and administrative business logic.

## 🚀 Key Features

* **🔐 JWT Authentication:** Implemented secure, token-based authentication using **Firebase Admin SDK**.
* **🛡️ Security First:** Replaced URL-based query parameters with server-side token verification for user-specific data.
* **💳 Payment Integration:** Seamless transaction handling via **Stripe API**.
* **📊 Business Analytics:** Custom API endpoints to calculate revenue, order stats, and category-wise distribution.
* **🎫 Coupon Management:** Advanced logic for discount validation, including expiry dates and usage limits.

## 🛠️ Tech Stack

* **Runtime:** Node.js
* **Framework:** Express.js
* **Database:** MongoDB
* **Auth:** Firebase Admin SDK
* **Payment:** Stripe API
